### 上手指南

###### 开发前的配置要求

1.按照requirements.txt中的包要求创建虚拟环境
2.Enjoy

###### **安装步骤**
1. Clone the repo

```sh
git clone https://github.com/shaojintian/Best_README_template.git
```
2. 确保您的环境中有python及node
3. 按照一般部署方法部署
### 开发的架构 

React + Ant Design + Django + MySQL

### 部署

Docker文件夹中含有docker-compose.yml，可以通过下列命令快速启动

```shell
docker-compose up
```
